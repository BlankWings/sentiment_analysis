项目在于解决京东apple手机评论的情感分析问题
根据类别，分类问题分为三种：
二分类：正向和负向，对应的评分为5,4和1（5,4为正向，1为负项）
三分类：好评，中评，差评，对应；评分分别为5,4;3,2；1（5,4好评，3,2中评，1为差评）
五分类：根据评分分为1,2,3,4,5等5个类别。

基本文件目录介绍：
主目录下有data文件夹，model文件夹，和code文件夹
data文件夹原始文件和处理后的文件
model文件夹用于处理训练模型
code文件夹是保存处理整个问题的程序

项目流程介绍：
1.对评论信息去重，保存为txt文本
2.对评论分词，保存为txt文本
3.对评论去停用词，保存为txt文本
4.将去停用词前后的数据，生成评论长度，并一起写入dataframe，保存为csv文本
5.可查看dataframe的统计信息
